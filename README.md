# Next.js Studio Project

This is a modern Next.js project built with React, Tailwind CSS, and various other UI libraries.

## 🚀 How to Install and Run

Since this is a Node.js/TypeScript project, it uses a `package.json` file instead of a Python `requirements.txt`. All the required "packages" and dependencies are listed in the `package.json` file!

To download all default dependencies, open your terminal in this directory and execute:
```bash
npm install or npm install --legacy-peer-deps

```

Once the installation finishes, you can start your local development server by executing:
```bash
npm run dev
```

Your project will then be available at [http://localhost:9002](http://localhost:9002).

## ⚙️ Environment Configuration

Make sure your `.env.local` file contains the appropriate environment variables. All logic and connections are handled within the `src/` directory.
